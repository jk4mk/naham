# NAHAM Design System

Community-driven cooking marketplace. **Light mode only.** Warm, clean, trustworthy.

## Colors
| Token | Hex | Use |
|-------|-----|-----|
| Primary (green) | `#2E7D32` | Growth, food, income – main actions, earnings |
| Secondary (purple) | `#6C5CE7` | Trust – secondary actions, accents |
| Background | `#FAFAF8` | Off-white scaffold |
| Card | `#FFFFFF` | White cards, soft shadow |
| Surface | `#F5F5F3` | Inputs, chips |
| Text primary | `#2D3436` |
| Text secondary | `#636E72` |
| Success | `#00B894` |
| Error | `#D63031` |

## Rules
- No dark mode
- No heavy gradients
- No glassmorphism
- Minimal shadows (card elevation 1)
- Soft rounded corners: 12–16 px
- Arabic-first layout support; large readable headings; avoid small text

## Usage
- `AppDesignSystem` – tokens (colors, spacing, radius, typography).
- `AppTheme.lightTheme` – Material theme using the design system.
